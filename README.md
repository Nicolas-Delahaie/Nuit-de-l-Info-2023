# Nuit_de_l_Info_2023
Projet dans le cadre de la Nuit de l'Info du 7-8 décembre 2023.

## Lancement Laravel
- lancer xamp (ou autre)
- Aller DANS le dossier laravel
- composer install
- php artisan key:generate
- configurer le .env généré avec les identifiants de sa bdd xamp
- php artisan migrate
- php artisan serve
